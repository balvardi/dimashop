# دیما شاپ - سیستم مدیریت فروشگاه آنلاین

## 🚀 معرفی
دیما شاپ یک سیستم مدیریت فروشگاه آنلاین پیشرفته و کامل است که با استفاده از تکنولوژی‌های مدرن و امن ساخته شده است. این سیستم قابلیت‌های متنوعی برای مدیریت کامل یک فروشگاه آنلاین ارائه می‌دهد.

## ✨ ویژگی‌های کلیدی

### 🛍️ مدیریت محصولات
- افزودن، ویرایش و حذف محصولات
- مدیریت دسته‌بندی‌ها و ویژگی‌ها
- سیستم تصاویر چندگانه
- مدیریت موجودی و قیمت‌ها
- سیستم تخفیف و کوپن

### 🤖 هوش مصنوعی (AI)
- تولید خودکار توضیحات محصولات
- نوشتن مقالات بلاگ با AI
- چت هوشمند پشتیبانی
- تحلیل تصاویر محصولات
- تولید محتوای SEO بهینه

### 🔐 امنیت پیشرفته
- فایروال امنیتی چندلایه
- محافظت در برابر حملات SQL Injection
- محافظت در برابر حملات XSS
- سیستم محدودیت نرخ درخواست
- مسدودسازی IP های مشکوک

### 📊 داشبورد مدیریتی
- آمار فروش و محصولات
- نمودارهای تحلیلی
- مدیریت سفارشات
- گزارش‌گیری جامع
- مدیریت کاربران

### 💳 سیستم پرداخت
- درگاه پرداخت زرین‌پال
- مدیریت سفارشات
- سیستم ارسال
- پیگیری سفارشات

### 📝 سیستم بلاگ
- مدیریت مقالات
- سیستم نظرات
- SEO بهینه
- دسته‌بندی محتوا

## 🛠️ تکنولوژی‌های استفاده شده

### Backend
- **PHP 8.0+** - زبان برنامه‌نویسی اصلی
- **MySQL/MariaDB** - پایگاه داده
- **PDO** - اتصال امن به پایگاه داده
- **Session Management** - مدیریت نشست‌ها

### Frontend
- **HTML5** - ساختار صفحات
- **CSS3** - استایل‌دهی
- **JavaScript ES6+** - تعامل کاربر
- **Bootstrap 5** - فریم‌ورک CSS
- **Tailwind CSS** - فریم‌ورک CSS مدرن
- **Chart.js** - نمودارها و گراف‌ها

### امنیت
- **CSRF Protection** - محافظت در برابر حملات CSRF
- **Input Validation** - اعتبارسنجی ورودی‌ها
- **SQL Injection Prevention** - پیشگیری از تزریق SQL
- **XSS Protection** - محافظت در برابر XSS
- **Rate Limiting** - محدودیت نرخ درخواست

### AI و API
- **Talkbot API** - سرویس هوش مصنوعی
- **GPT-4o-mini** - مدل زبانی
- **Image Analysis** - تحلیل تصاویر
- **Content Generation** - تولید محتوا

### ابزارها و کتابخانه‌ها
- **TinyMCE** - ویرایشگر متن پیشرفته
- **Font Awesome** - آیکون‌ها
- **Nucleo Icons** - آیکون‌های اضافی
- **Argon Dashboard** - قالب داشبورد

```

│   ├── assets/           # فایل‌های استاتیک
│   ├── includes/         # فایل‌های مشترک
│   └── pages/           # صفحات مختلف
├── api/                  # API های سیستم
├── config.php           # تنظیمات اصلی
├── db/                  # فایل‌های پایگاه داده
├── docs/                # مستندات
├── install/             # نصب کننده
├── lib/                 # کتابخانه‌های اصلی
├── logs/                # فایل‌های لاگ
├── sql/                 # فایل‌های SQL
├── theme/               # قالب‌های فروشگاه
└── uploads/             # فایل‌های آپلود شده
```

## 🚀 نصب و راه‌اندازی

### پیش‌نیازها
- PHP 8.0 یا بالاتر
- MySQL 5.7 یا بالاتر
- Apache/Nginx
- mod_rewrite فعال

### مراحل نصب
1. فایل‌ها را در پوشه وب‌سرور کپی کنید
2. به آدرس `http://your-domain/install/` بروید
3. مراحل نصب را دنبال کنید
4. تنظیمات پایگاه داده را وارد کنید
5. سیستم را راه‌اندازی کنید

## ⚙️ تنظیمات

### تنظیمات اصلی
- `config.php` - تنظیمات پایه
- `admin/pages/settings.php` - تنظیمات سیستم
- `admin/pages/ai-settings.php` - تنظیمات AI

### متغیرهای محیطی
- `DB_HOST` - آدرس پایگاه داده
- `DB_NAME` - نام پایگاه داده
- `DB_USER` - نام کاربری
- `DB_PASS` - رمز عبور
- `TALKBOT_API_TOKEN` - توکن API هوش مصنوعی

## 🔧 توسعه و سفارشی‌سازی

### افزودن ویژگی جدید
1. فایل جدید را در پوشه `lib/` ایجاد کنید
2. کلاس را در `config.php` بارگذاری کنید
3. صفحه مدیریت را در `admin/pages/` اضافه کنید
4. قالب فروشگاه را در `theme/default/` بروزرسانی کنید

### تغییر قالب
- فایل‌های قالب در `theme/default/`
- استایل‌ها در `theme/default/assets/css/`
- اسکریپت‌ها در `theme/default/assets/js/`

## 📚 API ها

### محصولات
- `GET /api/search.php` - جستجوی محصولات
- `GET /api/cart.php` - مدیریت سبد خرید

### احراز هویت
